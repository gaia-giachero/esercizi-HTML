# HTML Boilerplate
**This is a template HTML5.Thanks this boilerplate I can start whit a base code.**

# Characteristics
1. HTML5 with semantic elements;
2. Meta tag SEO: for optimization on browser web;
3. Responsive: thanks to meta viewport configured for mobile device;
4. Open graph: tag for share to social media.
